# Giggz
[giggz.mrammo.ca](https://giggz.mrammo.ca)

Digitally connecting small businesses through appointment bookings across all platforms.
Powered by Node.js, Express.js, React.js, and MongoDB.

## Alpha V0.1
Focusing on creating the fundamental back-end and app framework. Initial release will cater mainly towards barbershops in Windsor.
Future expansions will focus on expanding to a larger target audience.

## Dev Notes
* Mongodb and Redis server need to be running on ports 27017 and 6379 respectively.
* All authentication is bypassed when run in develoment
	* `NODE_ENV=production npm start` to run in production
